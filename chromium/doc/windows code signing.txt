***Signing with a self signed certificate***

To enable code signing create a selfsigned key and certificate in this way:

   "c:/Program Files (x86)/Windows Kits/8.1/bin/x64/makecert.exe" /r /pe /n "CN=<My> machine" /m 12 /sr CurrentUser /ss TestCertStore /sv f:/mykey.pvk f:/mykey.crt

   "c:/Program Files (x86)/Windows Kits/8.1/bin/x64/pvk2pfx.exe" /pvk f:/mykey.pvk /pfx f:/mykey.pfx /spc f:/mykey.crt

Create these enviroment variables:

  VIVALDI_SIGN_EXECUTABLE="c:/Program Files (x86)/Windows Kits/8.1/bin/x64/signtool.exe"
  VIVALDI_SIGNING_KEY="/f f:/mykey.pfx"

then rebuild after restarting the build utility

Setting up on autobuilder:

  "c:/Program Files (x86)/Windows Kits/8.1/bin/x64/makecert.exe" /r /pe /n "CN=Vivaldi testbuild" /m 12 /sr CurrentUser /ss TestCertStore /sv c:\bot\autobuilder.pvk c:\bot\autobuilder.crt
  "c:/Program Files (x86)/Windows Kits/8.1/bin/x64/pvk2pfx.exe" /pvk c:\bot\autobuilder.pvk /pfx c:\bot\autobuilder.pfx /spc c:\bot\autobuilder.crt

Environment variables:
  
  VIVALDI_SIGN_EXECUTABLE="c:/Program Files (x86)/Windows Kits/8.1/bin/x64/signtool.exe"
  VIVALDI_SIGNING_KEY="/f c:/bot/autobuilder.pfx"
